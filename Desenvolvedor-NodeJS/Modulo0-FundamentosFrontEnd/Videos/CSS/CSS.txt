:first-child  elemento que é o primeiro filho de seu pai
:last-child   elemento que é o último filho de seu pai
:empty        elemento que não possui filho
:not(E)       elemento não corresponde ao seletor E

ESPECIFICIDADE DE SELETORES - criterios de prioridade
1. declaracao dentro do atributo style
2. quantidade de seletores por ID da regra
3. quantidade de seletores por classe ou PSEUDOCLASSES da regra
4. quantidade de seletores por elemento da regra
5. ordem de declaracao da regra (última é aplicada)

PROPRIEDADE OVERFLOW - relativo ao comportamento do conteudo no box. se o conteudo for maior que o box
visible - padrao. o conteudo pode ser redenrizado fora do bbox 
hidden - conteudo é cortado
scroll - exibe o scroll bar sempre
auto - exibe o scroll bar se precisar



POSICIONAMENTO DE BOXES - BLOCK BOX
exibido em uma nova linha 
largura é 100% do espaço disponivel em seu container, por padrao
exemplo: h1, div, p, ul, li


POSICIONAMENTO DE BOXES - INLINE BOX 
exibido na mesma linha (pode ter quebra de linha se não houve espeço)
width e height : nao se aplicam 
margin, padding e border verticais se comportam diferente (não empurram os vizinhos)
exemplo: span, a, img, em, strong



PROPRIEDADE TEXT-ALIGN


PROPRIEDADE VERTICAL-ALIGN
controla o alinhamento vertical de inline box ou celulas de table
valores comuns::: baseline, middle, top, bottom



PROPRIEDADE DISPLAY - inline ou block 
por meio disto podemos dizer se o elemento vai ser BLOCK box ou inline box
display: block 
inline-block: se comporta como inline externamente, e como block internamente aceitando width e height. podemos usar isso quando queremos colocar um elemento block dentro de um elemento inline, devemos especificar este tipo de display sob pena de quebrar o layout


POSICIONAMENTO FORA DO NORMAL FLOW
controlado pela propriedade position
valor static é padrão(normal flow)
fora do normal FLOW
    relative
    absolute
    fixed
    sticky



POSITION: RELATIVE 
posicionado de acordo com normal flow, mas deslocado de sua posicao normal via propriedades top, right, bottom, left


POSITION ABSOLUTE 
posiciona fora do normal flow - nao ocupa espaco


POSITION FIXED 
posicionado fora do normal flow -  nao ocupa espacço
nao faz scroll na medida que rola a pagina





FLEXBOX
display: flex ou inline-flex 
flex -> comporta como block box externamente 
inline-flex -> comporta como inline box externamente
flex-direction: row ou column
